##description

本程序是一个类似与markdown的程序。用于对文本解析，并转换成html格式。

##command

    ./python_html.sh [file or directory]

###example

    ./python_html.sh readme
    
##instruction

>  # 一级标题
>  ## 二级标题
>  ### 三级标题
>  #### 四级标题
>  ##### 五级标题
  
>  \*...\*  匹配斜体
>  \*\*...\*\* 加粗
>  \> 注释
>  \\ 转义符号

>  http://xxx    网址
>  xxx@xx.xx    邮件地址
>
